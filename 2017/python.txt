	******* 1 *******
	if in __init__.py we say: import module1

	Instead, we should use: from . import module1

	import a module multiple times will not reload the module multiple times,
	reload function will reload the module again

	** 3 **
	when reload(package), it will run __init__.py in package
	when reload(module), it will run module.py
